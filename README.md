## Utilisation

## Opérations 

Il est possible de faire les même opérations de CRUD classique : 

| Opéaration | HTTP |
| ---------- | ----- |
| Création | POST |
| Récupération | GET |
| Modification | PUT |
| Suppresion | DELETE |
