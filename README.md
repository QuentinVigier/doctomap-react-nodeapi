json-server
Création d'une api json en local de type REST.

Installation
npm init -y pnpm i json-server SCRIPT dans package "start": "npx-json-server --watch db.json"

Utilisation
Les endpoints (pointd d'accès) sont définis dans le fichier db.json

GET

pnpm i express
pnpm i nodemon 
pnpm i ejs
node index.js
